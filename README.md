# MyTestMacro

A simple FreeCAD macro.
EuroRackForge — a FreeCAD macro for generating Eurorack facepanels.
## Installation

Install using FreeCAD Addon Manager, or copy `MyTestMacro.FCMacro` and `MyTestMacro.svg` into your FreeCAD macro directory.

## Usage

Open FreeCAD and run the macro from:

Macro → Macros…

## Files

- `MyTestMacro.FCMacro` - the macro
- `MyTestMacro.svg` - the icon
